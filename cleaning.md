# ハンズオン環境の削除

## 1. サービスの削除
 - サービスの画面から削除したいサービスを選択します
 - **アクション** ＞ **削除** をクリックします

 ![](./img/app_runner_delete.png)

 - 確認画面では指示に従って削除しましょう

 ![](./img/app_runner_delete_confirm.png)

## 2. GitHub接続の削除
 - GitHub 接続の画面から、今回作成した接続名を選択し、 **削除** をクリックします

![](./img/app_runner_github_delete.png)

 - 確認画面では指示に従って削除しましょう

![](./img/app_runner_github_delete_confirm.png)
